# 👑 Monarch Framework

Monarch is a modular CLI framework designed to manage and run tools in a structured environment.  
It provides a dynamic module system, automatic dependency installation, and a clean terminal interface.

The goal of Monarch is to provide a **simple and expandable toolkit framework** where tools can be added as modules without modifying the core system.

---

## ⚔ Features

- Modular plugin architecture
- Automatic module discovery
- Folder-based module organization
- Module metadata support
- Automatic dependency installation
- Built-in update system
- Uninstall support
- Dynamic banner engine
- Hidden command system
- Lightweight CLI framework

---
